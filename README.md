一个社区驱动的**优秀中文 Prompt 收集仓库**，专注于高质量、实用的中文提示词（Prompt），适用于 ChatGPT、Claude、Grok、Gemini、通义千问、Kimi、文心一言等大语言模型。

欢迎大家贡献、分享你的绝佳 Prompt，让中文 AI 用户玩得更溜！🚀

## 仓库目标

- 收集和整理**原创或优化过的优秀中文 Prompt**，覆盖各种场景。
- 提供结构化分类，便于快速查找和复制使用。
- 强调**实用性、可复现性**：每个 Prompt 最好附带简短描述、适用模型、示例输入/输出。
- 填补中文 Prompt 资源的空白（现有资源多为英文翻译版或零散分享）。
- 构建活跃社区，大家一起迭代优化 Prompt。

## 分类结构

Prompts 按文件夹组织，目前规划以下类别（后续可根据贡献扩展）：

- **日常助手**：生活助手、聊天、脑暴创意等（`/daily`）
- **写作创作**：文章写作、小说创作、诗词、营销文案等（`/writing`）
- **编程开发**：代码生成、调试、解释、算法等（`/coding`）
- **学习教育**：讲解知识、备考、出题、翻译等（`/education`）
- **专业领域**：法律、医疗、金融、设计、营销等（`/professional`）
- **图像生成**：Midjourney、Stable Diffusion、通义万相等中文提示词（`/image`）
- **角色扮演**：各种有趣角色、游戏化 Prompt（`/roleplay`）
- **高级技巧**：Chain of Thought、Few-Shot、Tree of Thoughts 等进阶 Prompt（`/advanced`）
- **其他**：不属于以上类别的优秀 Prompt, 可放入（`/misc`）.或者自建类别目录

每个类别下用 Markdown 文件或 JSON 格式存储，例如：

```markdown
### Prompt 标题（简短描述）

**适用模型**：ChatGPT / Claude / All

**Prompt 正文**：
```

## 如何贡献

非常欢迎大家贡献！即使是新手也没关系，一起让这个仓库变得更丰富。

1. **Fork** 本仓库。
2. 在对应分类文件夹下新增或编辑 Markdown 文件（推荐一个 Prompt 一个文件，便于管理）。
3. 严格遵守以上格式，确保 Prompt 是**中文为主**、高质量、可直接复制使用。
4. 提交 **Pull Request**，标题格式：`Add: [分类] Prompt标题`（例如 `Add: writing 爆款小红书文案生成器`）。
5. 在 PR 描述中说明 Prompt 的来源、灵感或测试效果（可选，但鼓励）。

**注意事项**：

- 避免提交低质量、纯翻译的英文 Prompt（除非显著优化过）。
- 不要提交违规、有害内容。
- 如果 Prompt 很长或有多个变体，可放在一个文件中。
- 我们会定期审合并优化结构。

**其他贡献方式**：

- 在 Issues 中提出新分类建议、报告问题。
- 在 Discussions 中分享 Prompt 使用心得、求推荐。
